# Wish List

## How to run locally

1. Install dependencies:
```bash
npm install
```

2. Start JSON Server (in one terminal):
```bash
npm run server
```
Server will run on `http://localhost:3000`

3. Start dev server (in another terminal):
```bash
npm run dev
```

4. Open browser and go to `http://localhost:5173`

Done! You can now use the app.
